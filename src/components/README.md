# مكونات واجهة المستخدم

هذا المجلد يحتوي على مكونات UI القابلة لإعادة الاستخدام.

## التنظيم

```
├── common/           # المكونات الأساسية (الأزرار، المدخلات، إلخ)
├── forms/            # مكونات النماذج
├── layout/           # مكونات التخطيط
├── cards/            # مكونات البطاقات
└── navigation/       # مكونات التنقل
```

## إرشادات

- كل مكون يجب أن يكون في مجلد منفصل
- استخدم متغيرات CSS المعرفة في ملف styles
- اتبع مبادئ التصميم المتجاوب
- وثق المكونات الجديدة
